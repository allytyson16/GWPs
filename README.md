# GWPs

Overview
This repository contains the implementation and analysis for DP. The project focuses on:

- Pricing European and American options using binomial and trinomial trees.

- Validating option prices with put‑call parity.

- Computing Greeks (Delta, Vega) and performing dynamic delta hedging.

- Comparing European, American, and Asian put options.

- Visualising convergence and option price behaviour under different parameters.

All code is written in Python and run in Google Colab / Jupyter notebooks.
